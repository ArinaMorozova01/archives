# Домашняя работа №3. Оформление репозитория на GitHub

## Описание
Моя тематическая [коллекция](https://drive.google.com/drive/folders/1LT7-yLDEwpThNNgjD--Kiaga-GbYtHbn) - это сайты с полезными материалами о шпицах. Я очень люблю померанских шпицев, но пока, к сожалению, не могу завести, поэтому создам эту коллекцию от лица гипотетического владельца собаки. Здесь я сохранила небольшие блоги с полезными статьями о шпицах. 
На мой взгляд архивировать такие небольшие сайты - актуальная задача, т.к. их владельцы обычно простые люди, которые могут прекратить поддержку сайта по разным причинам (например, сейчас стало менее выгодно содержать сайты-блоги, т.к. к ним обращается все меньше людей, предпочитая видеоконтент). 
Несколько лет назад я часто читала один такой блог, но в какой-то момент потеряла его и так и не смогла больше найти, вероятнее всего владелец просто его закрыл. Поэтому стоит сохранять понравившиеся блоги, чтобы в будущем, когда у меня появится собака, у меня уже будет собрана коллекция со всем необходимым.

В моей коллекции 3 сайта с полезной информацией по уходу за шпицами, которую хотелось бы сохранить для себя и 1 сайт питомника, откуда родом сам шпиц, в разделе выставки есть его фотографии как участника и победителя выставок, упоминания о которых, мне как гипотетическому хозяину хотелось бы сохранить.

## Сайты: 
https://shpicdog.ru - много о здоровье шпицев

https://spitz-sobaka.ru - статьи про выведение новых пород от помеси со шпицами

https://oshpice.ru - статьи о видах, различии окрасов и стандартах породы

http://niksend.com - питомник с отчетами с различных выставок

## Метод и технология сбора архивов
1. Все архивы были собраны при помощи специальной утилиты [wpull](https://wpull.readthedocs.io/en/master/), которая позволяет архивировать сайты с задаваемыми параметрами для архивации
2. Для воспроизведения warc-файлов с архивами сайтов использовался сервис [Replay Webpage](https://replayweb.page)
3. Для анализа полученных архивов была использована утилита [metawarc](https://github.com/datacoon/metawarc)
4. Для оценки архивируемости сайтов был использован сервис [ArchiveReady](https://archiveready.com)
